Primeira versão da plataforma de aprendizado sobre github

Aqui você pode perceber a importância de escrever a mensagem na hora do commit, pois assim é possível saber exatamente sobre o que se trata cada commit, inclusive o da primeira linha que foi comitado na aula anterior como exemplo. Boas descrições são essênciais.

Existem informações importantes em cada um deles, como quem fez o commit (já que é possível ter mais de uma pessoa trabalhando no mesmo código), email, quando e inclusive o identificador destacado em amarelo, uma hash única para cada commit, importante para nosso controle de versionamento que depois falaremos.

Podemos utilizar também 'git log --author='Nome' para pegar todos os commits de um determinado programador.

@User
MINGW64
~/Git Directory
(master)
$ git log --author='Lucas'
