Iniciando repositório

Iniciar um novo repositório é muito simples, mas antes de criar, você precisa abrir o git bash no diretório que você deseja iniciar o repositório.

Feito isso, tudo o que você necessita fazer é o seguinte comando:

@User
MINGW64
~/Git Directory
$ git init

Ao iniciar, você perceberá que na linha seguinte aparecerá em azul, a branch que você está (master).

$ git init
@User
MINGW64
~/Git Directory
(master)

Depois falaremos mais sobre o que são branches.

Digitando 'ls -la', você conseguirá ver todos os diretórios e arquivos que o Git está enxergando no diretório iniciado.

Abaixo você verá um exemplo de como isso pode aparecer pra você. Os arquivos que você verá são os atuais arquivos deste site no momento em que eu estou produzindo agora.
